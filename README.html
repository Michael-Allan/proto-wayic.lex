wayic.lex - Dictionary of wayical terms

Serving

Special response headers are needed for serving term documents. These are the documents without an extension on their file names. Appropriate headers for serving them are:

  • Content type: application/xhtml+xml
  • Access-Control-Allow-Origin: *

See the working example of a header configuration for Apache httpd.

The Access-Control-Allow-Origin header grants to programmatic requesters originating from all network domains (*) permission to read this dictionary’s term documents. Such requesters include the content importers both of other dictionaries published at remote locations, and of copies of the present dictionary stored in local file systems (null domain) and viewed in a browser under a ‘file’ scheme.
