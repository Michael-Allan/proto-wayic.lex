wayic.lex - Dictionary of wayic terms

When serving this project on the Web, or other projects that use it to define terms, you will want to serve the term documents with appropriate response headers. (The term documents are those which are stored in files without a filename extension.) Appropriate response headers are:

  • Content type: application/xhtml+xml
  • Access-Control-Allow-Origin: *

The purpose of allowing universal access (*) is that term documents located elsewhere (including those located by ‘file’ scheme URLs) may require access in order to resolve their content importers. See also the configuration example for Apache httpd.
