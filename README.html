wayic.lex - Dictionary of wayical terms

Serving

Term documents need special response headers. These are the documents without an extension on their file names. Appropriate headers for serving them are:

  • Content type: application/xhtml+xml
  • Access-Control-Allow-Origin: *

See the working example of a header configuration for Apache httpd.

The Access-Control-Allow-Origin header grants universal access (*) to the term documents. This enables content importers that were loaded into the user’s browser from a foreign network domain, or a ‘file’ scheme, to import them. In other words, it enables Web publishers to reuse the definitions in dictionary compilations of their own, and enables Web developers to test the compilations directly from their local file systems.
