wayic.lex - A dictionary of wayics

When serving this project on the web, or other projects that define words, you’ll want to serve the word files with appropriate response headers. The word files are the ones without a filename extension. The appropriate headers are:

  • Content type: application/xhtml+xml
  • Access-Control-Allow-Origin: *

The purpose of allowing universal access (*) is that word files served elsewhere, including those of local ‘file’ scheme origin, may require access in order to resolve their content insertion links. See also the configuration example for Apache httpd.
